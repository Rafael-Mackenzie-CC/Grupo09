As chamadas de sistema ao SO ocorrem quando o programa chama as funções fopen(), fgetc(), fputc() e fclose(). 
Essas funções são definidas na biblioteca padrão do C e são responsáveis por acessar o sistema de arquivos do SO para abrir, ler, escrever e fechar arquivos.
